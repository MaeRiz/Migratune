Comment obtenir tes headers ?
  1. Ouvre music.youtube.com et connecte-toi
  2. Appuie sur F12 → onglet Réseau / Network
  3. Dans la barre de filtre, tape browse ou search
  4. Fais une recherche sur YouTube Music pour déclencher une requête API
  5. Clique sur la requête /youtubei/v1/search ou /youtubei/v1/browse
  6. Dans le panneau de droite → En-têtes de requête → vérifie qu'il y a un header authorization: SAPISIDHASH …
  7. Copie tous les en-têtes de requête et colle-les ci-dessous

Déjà connecté ? Si browser.json existe déjà, laisse le champ vide.
